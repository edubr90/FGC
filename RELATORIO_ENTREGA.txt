6. RESUMO DO PROJETO
================================================================================

Tipo: API RESTful
Linguagem: C# (.NET 8)
Framework: ASP.NET Core 8
Banco de Dados: SQL Server (LocalDB)
Padrão de Arquitetura: Camadas (Domain, Application, Infrastructure, API)

Objetivo Geral:
Desenvolver uma API robusta e segura para gerenciar um sistema de autenticação
de usuários e um catálogo de jogos, demonstrando a aplicação de padrões SOLID,
clean code e boas práticas de desenvolvimento em .NET.

Funcionalidades Principais:
  ✓ Autenticação e autorização com JWT
  ✓ Gerenciamento de usuários (Create, Read, Update, Delete)
  ✓ Gerenciamento de jogos (Create, Read, Update, Delete)
  ✓ Sistema de roles (Admin, Player)
  ✓ Validação robusta de entrada
  ✓ Tratamento de erros centralizado
  ✓ Injeção de dependência

Tecnologias Utilizadas:
  - .NET 8 SDK
  - ASP.NET Core 8
  - Entity Framework Core 8
  - SQL Server LocalDB
  - JWT (JSON Web Tokens)
  - Swagger/OpenAPI
